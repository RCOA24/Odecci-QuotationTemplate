# Odecci Quotation Template

This project is a reusable quotation/invoice template created for Odecci Solutions Inc. to generate professional client quotes for web and design services.  
The layout was implemented in HTML and SCSS, matching Odecci’s branding and supporting itemized services, rates, and totals. 

---

## About the Template

The quotation document is designed for small agencies or service providers who need a clean, printable quote they can quickly customize per client.  
It includes sections for company details, client details, service breakdown, payment terms, and contact information. 

---

## Structure

The template contains the following key sections: 

- **Company Header** – Odecci Solutions Inc. name, address, and primary contact email.  
- **Quotation Title Block** – Prominent “QUOTATION” label plus quote number, sales order number, date, and approval status.  
- **Bill To Panel** – Client name, address, and billing details.  
- **Services Table** – Description, hours/quantity, rate, and amount for each service line (e.g., web design, logo design, SEO, content writing).  
- **Totals Summary** – Subtotal, tax information, total, and amount due.  
- **Terms of Payment & Notes** – Space for payment terms and additional notes or instructions for the client.  
- **Footer Contacts** – Company website, billing email, and phone number, plus registration identifiers.  

---

## Tech Stack

- **Markup:** HTML  
- **Styling:** SCSS (compiled to CSS)  
- **Output:** Printable, PDF-ready quotation layout for client-facing documents. 

---

## Usage

1. Open the HTML file in your code editor.  
2. Update the **Bill To** section with client information.  
3. Modify the **Services** table rows with relevant descriptions, hours/qty, and rates.  
4. Adjust totals and tax calculations as needed or integrate with your back-end logic.  
5. Export to PDF via the browser print dialog or your application’s PDF generation pipeline.
